# starter

a hexo starter stack for hanley wood's interactive team

## getting started

```
$ git clone https://github.com/hw-interactive/starter.git
$ cd starter
$ npm install
```

## commands

* `$ npm start` -> to open dev server
* `$ npm run deploy` -> to deploy as configured in `./_config.yml`
